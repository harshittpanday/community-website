# UNBOUND Community Website

A community-focused website project designed for an online gaming community.  
This project was created to improve community accessibility, branding, and user experience through organized navigation and public information pages.

## 🌐 Features
- Homepage layout and design
- Navigation improvements
- Community information pages
- Branding and visual consistency
- User-friendly structure

## 🛠️ Skills Used
- HTML
- CSS
- Website Management
- Community Branding
- Basic UI Design

## 🎯 Purpose
The goal of this project was to create a clean and accessible web experience for community members while improving overall presentation and organization.
